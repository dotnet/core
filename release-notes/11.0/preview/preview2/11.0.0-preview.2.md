# .NET 11 Preview 2 Binaries

.NET 11 Preview 2 installers and binaries can be downloaded for the following product versions:

- `.NET 11.0.0 Preview 2`
- `.NET SDK 11.0.100-preview.2.BUILD`

See [README.md](README.md) for .NET 11 release notes and discussions.

The latest .NET releases are always listed at [.NET Releases](../../README.md).

## Downloads

Download links and checksums will be added closer to release.

## Container Images

[.NET container images](https://github.com/dotnet/dotnet-docker/blob/main/README.md) are updated for preview releases.

## Visual Studio Compatibility

.NET 11 is compatible with the latest Visual Studio preview. Users will need to download and install the .NET 11 runtime/SDK standalone.
